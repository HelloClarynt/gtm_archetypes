# Clarynt × PwC GTM Archetypes Workspace

A live, multi-user workspace for working through FMCG growth archetypes
in joint Clarynt + PwC sessions.

## Live site

https://helloclarynt.github.io/gtm_archetypes/fmcg-archetypes-live.html

## For editors

On first open, the app asks for:
- An **Apps Script URL** — you'll have been sent this separately
- **Your name** — use the format `Name (Org)`, e.g. `Jane Doe (PwC)`

Both are saved locally in your browser. Click "Continue" and you're in.

## Stack

Single HTML file. Backend is a Google Apps Script Web App persisting
to a Google Sheet. No build step, no install, no auth for editors.
